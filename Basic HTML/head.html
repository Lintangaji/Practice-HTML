Ini adalah penjelasan tentang tag head

Elemen <head> dalam HTML memiliki beberapa fungsi penting. Berikut adalah beberapa fungsi utama dari elemen <head>:


  1. Metadata: Elemen <head> digunakan untuk menyimpan metadata atau informasi tentang dokumen HTML yang tidak terlihat oleh pengguna, tetapi penting untuk keperluan teknis atau SEO. Contoh metadata yang umum termasuk judul halaman <title>, deskripsi (<meta name="description" content="...">), dan kata kunci (<meta name="keywords" content="...">).

  2. Penyisipan CSS dan JavaScript: Elemen <head> digunakan untuk menyertakan berkas CSS eksternal (<link rel="stylesheet" href="...">) dan skrip JavaScript eksternal (<script src="..."></script>) ke dalam dokumen. Ini memungkinkan Anda untuk mengatur tampilan dan perilaku halaman dengan menghubungkan file eksternal.

  3. Perubahan ikon halaman: Dalam elemen <head>, Anda dapat menentukan ikon atau favicon yang akan ditampilkan di tab browser atau bookmark. Ini dilakukan dengan menggunakan elemen <link> dengan atribut rel="icon" atau rel="shortcut icon".

  4. Menetapkan karakter encoding: Elemen <head> digunakan untuk menetapkan karakter encoding atau set karakter yang digunakan dalam dokumen HTML. Ini dilakukan dengan menggunakan elemen <meta charset="...">.

  5. Hubungan dengan stylesheet alternatif dan stylesheet cetak: Dalam elemen <head>, Anda dapat menyertakan link ke stylesheet alternatif untuk perangkat atau keadaan khusus (<link rel="alternate stylesheet" href="...">). Anda juga dapat menentukan stylesheet yang akan digunakan saat halaman dicetak (<link rel="stylesheet" media="print" href="...">).

Elemen <head> biasanya tidak terlihat oleh pengguna secara langsung, tetapi berperan penting dalam memberikan informasi teknis, menghubungkan berkas eksternal, dan mengatur metadata yang relevan dengan dokumen HTML.
