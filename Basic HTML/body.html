Ini adalah penjelasan tag body html


Tag <body> dalam HTML digunakan untuk menentukan isi dari halaman web yang akan ditampilkan kepada pengguna. Ini merupakan salah satu tag wajib dalam struktur HTML dan menempati area utama dari halaman web.


Berikut adalah beberapa fungsi utama dari tag <body>:

  1. Menyediakan konten utama: Semua konten yang ingin ditampilkan kepada pengguna, seperti teks, gambar, video, elemen formulir, dan lainnya, ditempatkan di dalam tag <body>. Isi yang diberikan di dalam tag <body> akan ditampilkan di jendela browser.
  2. Memisahkan bagian utama dari bagian kepala: Dalam HTML, informasi seperti judul halaman, meta tag, link ke file CSS, dan script JavaScript ditempatkan dalam tag <head>. Bagian ini terpisah dari bagian <body>. Dengan memisahkan bagian kepala dan bagian tubuh halaman dengan menggunakan tag <body>, itu memungkinkan browser dan mesin pencari untuk membedakan dan menginterpretasikan informasi yang berbeda dengan cara yang sesuai..
  3. Menyediakan skrip JavaScript: Jika ada kode JavaScript yang ingin dijalankan saat halaman dimuat, tag <body> adalah tempat yang tepat untuk menempatkannya. Ini memungkinkan pengembang web untuk menentukan fungsi-fungsi JavaScript yang akan dijalankan atau melakukan manipulasi pada elemen-elemen HTML yang ada di dalam tag <body>.
  4. Menentukan perilaku halaman: Tag <body> dapat mengandung atribut-atribut seperti onload, onunload, onresize, dan lainnya. Atribut-atribut ini memungkinkan penentuan perilaku khusus saat halaman dimuat, ditutup, atau mengalami perubahan ukuran.
  5. Menyediakan struktur halaman: Tag <body> juga dapat digunakan untuk membagi halaman menjadi bagian-bagian yang terpisah. Misalnya, dengan menggunakan tag <header>, <main>, <footer>, dan lainnya di dalam tag <body>, pengembang web dapat menentukan bagian-bagian halaman seperti kepala, konten utama, dan footer dengan lebih terstruktur.

Dalam rangka mengembangkan halaman web yang efektif dan semantik, tag <body> memainkan peran penting dalam menentukan konten dan perilaku halaman yang akan dilihat oleh pengguna.
