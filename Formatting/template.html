Definisi dan Penggunaan :

<template> digunakan sebagai wadah untuk menyimpan beberapa konten HTML yang disembunyikan dari pengguna saat halaman dimuat.

Konten di dalam tag <template> dapat dirender nanti menggunakan JavaScript.

Anda dapat menggunakan tag <template> jika Anda memiliki beberapa kode HTML yang ingin digunakan berulang-ulang, tetapi tidak sampai Anda meminta untuk menampilkannya. Untuk melakukan ini tanpa tag <template>, Anda harus membuat kode HTML dengan JavaScript untuk mencegah browser merender kode tersebut.
